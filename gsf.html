Galactic Structure Finder (gsf)

Galaxies are usually classified according to their appearance as either late type (barred or non-barred spirals and irregulars) or early type (ellipticals). Lenticular galaxies represent the transition between these two classes. These classes can be understood as forming an evolutionary sequence. Ellipticals are older and less organised internally due to mergers, whereas spirals are younger and have distinct features such as disks, bulges, bars and spiral arms. Spirals are also thought to have experienced a less violent merger history. If we can causally link the presence of substructures (and their properties) to specific events and their timings, we can effectively build a timeoscope to look backwards in time at observed galaxies. Additionally, specific features could offer insights into the surrounding dark matter. To this end, I have developed an analysis code that can segment simulated galaxies into physically meaningful components.

The first version of Galactic Structure Finder (gsf), which is mainly written in Python, uses direct N-body summation in Fortran-OpenMP to accurately compute gravitational forces. It can identify dynamical structures such as thin/thick disks, bulges, spheroids and stellar halos within a fixed 3D angular momentum-energy space. The rationale behind this code was the need for a data-driven galaxy segmentation instead of a model-driven segmentation, where restrictive and poorly founded physical distribution functions would have to be assumed for the various components. After experimenting with various clustering algorithms, I opted for Gaussian Mixture Models (GMMs) and developed the code as an advanced usage wrapper for the GMM class in the open-source library scikit-learn.

The second version of the code ( gsf2) goes a long way in the direction of automating the whole galaxy analysis process. The code now has two information criteria that can be used to select an optimal number of components, a naming scheme based on physical properties and the ability to select from an infinite range of feature spaces. It is the latter code feature in particular that has enabled us to devise a method of isolating galaxy bars, which are notoriously difficult to separate from the other components.



Segmentation of a barred galaxy from the TNG50 simulation using gsf2. The bar constitutes 36% of the total mass and appears significantly asymmetric in the face-on projection (b/a = 0.4). Separating the bar enables us to easily estimate properties that are difficult to measure, such as the pattern speed Ωb. Adapted from the paper just submitted to MNRAS.
